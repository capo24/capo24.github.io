冒泡排序

思想：相邻元素两两比较，若为逆序则交换次序，得出最大(最小)的元素，将其移动到最终位置。

快速排序

使用分治思想，递归排序元素。

以序列首元素为key，高端下标high从末端往前查找比key小的元素，将查找到的赋值给低端下标low；低端下标low从首端往后查找比key大的元素，将查找到的赋值给高端下标high；若首末端查找的下标重合，将key放到低端下标low的位置。注意，高端下标先行。

算法实现

void QucikSort(int a[],int low, int high){
    int l = low,h = high;
    int temp = a[low];
    while (l<h)
    {
        while (l<h&&temp<a[h])
            h--;
        if(l<h){
            a[l] = a[h];
            l++;}
        while (l<h&&temp>a[l])
            l++;
        if(l<h){
            a[h] = a[l];
            h--;}
        a[l] =temp;
        if(low < l)QucikSort(a, low, l-1);
        if(l < high)QucikSort(a, h+1, high);
    }
}
